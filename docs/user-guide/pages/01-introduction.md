# Introduction to Isometry

**Isometry** is an isometric pixel art RPG framework built with Godot. It provides a complete, data-driven system for creating custom campaigns with multiplayer support, AI behaviors, and rich gameplay mechanics.

## What is Isometry?

Isometry is both a game engine and a platform. Campaign creators design worlds using JSON configuration files, while players experience those worlds through the Isometry client. No programming is required to create campaigns.

## Key Features

- **Isometric Grid-Based Maps** - Beautiful diamond-layout isometric worlds
- **Multiplayer Networking** - Host, join, or run dedicated servers via ENet UDP
- **Data-Driven Campaigns** - Define everything in JSON, no code required
- **AI Behavior System** - Goal-based NPC strategies and patrol routes
- **Skill & Action System** - Up to 9 hotbar skills with conditional logic
- **Resource Management** - Health, mana, gold, and custom resources with triggers
- **Vision & Perception** - Line-of-sight, fog of war, and stealth mechanics
- **Chat System** - 8-channel multiplayer chat
- **Gamepad Support** - Full controller mapping

## System Requirements

- **OS**: Linux (x86_64), Windows (x86_64), or macOS
- **Disk**: ~200MB for the engine, plus campaign data
- **RAM**: 512MB minimum
- **Network**: Internet connection for multiplayer (UDP port 5000 default)
